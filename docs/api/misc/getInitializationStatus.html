getInitializationStatus

Use this function to get each mediation adapter's initialization status.

Usage exampleâ€‹

import AdMob from '@react-native-admob/admob';

export default function App() {
  useEffect(() => {
    const getStatus = async () => {
      const initializationStatus = await AdMob.getInitializationStatus();
      console.log(initializationStatus);
    };

    getStatus();
  }, []);

  return (/* Your App code */)

Returnsâ€‹

The function returns Promise<InitializationStatus[]> which is Promise of each mediation adapter's initialization status.

Type
Promise<InitializationStatus[]>

Properties of InitializationStatus:

NameTypeDescription
namestringName of the adapter.
descriptionstringDescription of the status.
isReadybooleanWhether the adapter is ready.
